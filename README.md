# 📰 Live News Feed

A modern and responsive news web application that fetches real-time news articles using the NewsData.io API. Users can browse the latest headlines by category, search for specific topics, and read full articles directly from the source.

## 🌐 Live Demo

https://ayeshaaamirsheikh1-design.github.io/Live-News-Feed/

---

## ✨ Features

* 🔥 Live news updates using NewsData.io API
* 📂 Category-based news filtering

  * General
  * Technology
  * Business
  * Sports
  * Health
  * Science
* 🔍 Search news by keywords
* 📰 Dynamic news cards
* 📅 Formatted publication dates
* 🖼️ Fallback image support for missing images
* ⏳ Loading spinner while fetching data
* ⚠️ Error handling for failed API requests
* 📱 Responsive design for desktop and mobile devices
* 🔗 Direct links to original news articles

## ⚙️ How It Works

1. The application fetches news data from the NewsData.io API.
2. Users can select different categories to filter news.
3. Search functionality allows finding articles on specific topics.
4. News cards are dynamically generated using JavaScript.
5. Each card displays:

   * News image
   * Source name
   * Headline
   * Description
   * Publication date
   * Read More link

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ayeshaaamirsheikh1-design/Live-News-Feed.git
```

Navigate to the project folder:

```bash
cd Live-News-Feed
```

Open `index.html` in your browser.

---

## 🔑 API Used

NewsData.io

To use your own API key:

1. Create an account on NewsData.io.
2. Generate an API key.
3. Replace the API key in `script.js`.

```javascript
const API_KEY = 'YOUR_API_KEY';
```
